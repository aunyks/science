My (Small) Research Projects

When I have free time, I like to use code to research and simulate what I read or inquire.

  • - Quandary

    • A file encryption tool. Quandary is a GoLang command line tool that uses PBKDF2 to derive a key to be used for symmetric encryption via AES and file integrity assurance via HMAC. The resulting ciphertext of the encrypted file is stored in a new, user-specified file with the hash of the ciphertext and the salt used to derive the key.

  • - Create Bitcon Wallet

    • A tiny Bitcoin wallet generator. Create-BTC-Wallet is a GoLang command line tool that implements the Bitcoin protocol's standard of wallet generation. Because there doesn't yet exist a native GoLang implementation of the secp256k1 elliptic curve, however, Create-BTC-Wallet's generated addresses are not compliant with the Bitcoin protocol and will not work in practical settings. To suppress this fact, generated addresses are redacted and not shown to the user; private keys are still shown, since they still follow the Bitcoin protocol's Wallet Import Format and can be used practically.

  • - Newton

    • A microservice for symbolic math computation. Newton is a Node.js RESTful API that can perform symbolic operations on expressions provided via its endpoints. The API uses an Abstract Tree representation of the given expressions to perform computationally difficult operations such as differentiation and integration, along with high order operations that employ the two aforementioned operations.
